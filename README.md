commands rules:
- clear:
    only 1 player and max 3 slots.
    dont clear everyones inventory.

- damage and effect:
    dont apply it too often.
    no instant kill.
    no infinite effects.

- fill, clone and setblock:
    dont spam it.
    dont place command blocks etc.
    dont fill max blocks.
    dont grief with this.

- forceload:
    max 3 chunks per players datapack.

- kill:
    dont spam it.
    only kill nonplayers.

- give and item:
    dont give OP or creative items.

- msg, teammsg, tell, tellraw, title and say:
    dont spam it.
    no links.

- particle and summon:
    dont spam it.

- spreadplayers:
    dont use it on players.

- teleport and tp:
    dont tp everything to 1 player.
    dont tp entities that will kill the player.

- weather:
    dont set the weather to your liking.

- gamemode:
    only adventure and survival are allowed.
